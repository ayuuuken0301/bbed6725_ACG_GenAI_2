# GenAI in ACG Industry Website

A modern, interactive website exploring the impact of Generative AI on the Anime, Comic, and Games (ACG) industry.

## Features

- **4 Main Tabs**: Introduction, Benefits, Risks, and Mitigation Measures
- **Interactive Value Chain Stepper**: Click on each stage (Upstream, Midstream, Downstream) to see detailed explanations
- **Turing Test Activity**: Interactive quiz to test if users can distinguish between AI-made and human-made anime art
- **Modern UI/UX**: Beautiful gradient design with smooth animations and responsive layout
- **Reference Links**: All examples include proper citations and links to sources

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Interactive functionality (tabs, stepper, Turing test)

## How to Use

Simply open `index.html` in a web browser. No build process or server required - it's a static website that works offline.

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge).

1. **Introduction**: Overview of ACG industry and value chain
2. **Benefits**: Production speed improvements, cost reduction, and hyper-personalization
